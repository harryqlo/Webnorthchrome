## 🆘 Tengo un Problema

### Opción 1: Rápida (1 minuto)
```bash
F5          ← Recarga la página
Ctrl+Shift+Del  ← Limpia caché
F5          ← Recarga nuevamente
¿Funciona?  ← Sí = Problema resuelto
```

### Opción 2: Debug (5 minutos)
```bash
F12                          ← Abre consola
localStorage.clear()        ← Limpia configuraciones
F5                          ← Recarga
SettingsManager.settings    ← Ver configuración
¿Funciona?                  ← Sí = Problema resuelto
```

### Opción 3: Experto (10 minutos)
```
1. Lee: SOPORTE_TECNICO.md
2. Busca: Tu problema específico
3. Sigue: Los pasos indicados
4. ¿Funciona?  ← Sí = Problema resuelto
```
